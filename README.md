# 🎮 RPG Quest To-Do (Game Version)

Turn your daily tasks into an exciting RPG experience ⚔️

## 🚀 Overview

**RPG Quest To-Do** is a gamified productivity web app where users complete tasks as quests, earn XP and level up. Instead of a traditional to-do list, this project makes productivity fun, engaging, and rewarding.

Inspired by gamified task systems like Habitica, this app uses game mechanics to motivate users to stay consistent and productive. 

---

## ✨ Features

* 🎯 **Tasks as Quests** – Add and complete tasks like missions
* ⭐ **XP System** – Earn experience points for completing tasks
* 📈 **Level Progression** – Level up as you gain XP
* 🏆 **Titles System** – Unlock ranks like Beginner, Executor, Master
* 🔥 **Daily Streak** – Stay consistent and build streaks
* 📊 **XP Progress Bar** – Visual progress tracking
* 🎉 **Popups & Feedback** – Instant satisfaction
* 💾 **Local Storage** – Saves your progress automatically

## ⚙️ How It Works

1. Add a task (quest)
2. Complete the task
3. Earn XP ⭐
4. Level up and unlock new titles 🏆

---

## 🧠 Concept

This project is based on **gamification**, a technique where game mechanics are used to improve user engagement and motivation.

Instead of:

> ❌ “I have to do tasks”

It becomes:

> ✅ “I want to complete quests and level up”
